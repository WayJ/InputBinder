# InputBinder
manager form activity(View,Model,WebApi),quick develop.
MVVM模式中的VM层框架。
一个能够快速开发form表单页面的工具，实现了多层关系绑定。且自带了基础样式，快速搭建复杂且数目较多的form表单页面。

- input  :  layout,实体类注解配置/xml配置,数据
- output :  用户输入完成后的数据-存储了键值对的map。用于发送post请求。

---------

#### 支持的form表单的input类型
* 文本输入框
* 按钮
* 日期选择
* 勾选框
* 单选框
* 多选框
* 支持自行扩展
